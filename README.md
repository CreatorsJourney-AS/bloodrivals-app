# BloodRivals — Android downloads

**BloodRivals** is a location-based mobile game: claim real-world spots as properties,
upgrade them, earn coins, and compete with your crew. This repository hosts the public
**Android builds** so you can try the game without the Play Store.

> The game source code lives in a private repository. This repo only holds release APKs.

## Download & install

1. Go to **[Releases](../../releases/latest)** and download `bloodrivals-android.apk`.
2. On your phone, open the file. Android will ask you to allow installs from your
   browser / file manager ("Install unknown apps") — enable it for that app, then install.
3. Open **BloodRivals** and sign in or create an account.

**Requirements:** Android 6.0 (SDK 23) or newer.

This build connects to the live BloodRivals servers, so you're playing the real game
alongside everyone else.

## Updating

New versions are published here as new releases. Because every build is signed with the
same key, you can install an update straight over your existing copy — no need to
uninstall first.

## Safety

- The APK is built and signed by our CI from the official source; the signing certificate
  is consistent across every release.
- Only install BloodRivals from this repository (or, once available, the Google Play Store).
